# bengal-unlearning — SHRED project page

Project page for **SHRED: Retain-Set-Free Unlearning via Self-Distillation with Logit Demotion**.

Live: https://zizhao-hu.github.io/bengal-unlearning/

SHRED is a retain-set-free LLM unlearning method, implemented for the
[OpenUnlearning](https://github.com/locuslab/open-unlearning) framework.
